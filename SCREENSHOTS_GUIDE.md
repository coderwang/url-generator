# 商店截图准备指南

为了在 Chrome Web Store 展示你的插件，你需要准备高质量的截图。

## 📸 截图要求

### 尺寸要求

- **推荐尺寸**: 1280x800 或 640x400
- **最小尺寸**: 640x400
- **最大尺寸**: 2400x1500
- **格式**: PNG 或 JPEG
- **数量**: 至少 1 张，最多 5 张

## 🛠️ 如何制作截图

### 方法 1: Chrome 开发者工具（推荐）

1. 打开插件 popup
2. 右键点击 popup → 选择"检查"
3. 在 DevTools 中：
   - 点击左上角的设备切换按钮
   - 设置尺寸为 1280x800
4. 使用 Chrome 内置截图：
   - Cmd+Shift+P (Mac) 或 Ctrl+Shift+P (Windows)
   - 输入 "Capture screenshot"
   - 选择 "Capture node screenshot"

### 方法 2: 系统截图工具

**macOS:**

```bash
# 截取指定区域
Cmd + Shift + 4

# 截取整个窗口
Cmd + Shift + 4，然后按空格键
```

**Windows:**

```bash
# 使用截图工具
Win + Shift + S
```

### 方法 3: 使用在线工具调整尺寸

如果截图尺寸不符合要求，可以使用这些工具调整：

- [TinyPNG](https://tinypng.com/) - 压缩 PNG
- [Squoosh](https://squoosh.app/) - 调整尺寸和压缩
- Photoshop / GIMP - 专业图像编辑

## 🎨 截图最佳实践

### 视觉效果

- ✅ 使用真实的数据和 URL
- ✅ 确保文字清晰可读
- ✅ 保持 UI 完整，不要截断
- ✅ 使用一致的浏览器主题
- ✅ 显示实际使用场景

### 内容建议

- ✅ 展示核心功能
- ✅ 突出独特卖点
- ✅ 显示用户价值
- ✅ 保持简洁清晰
- ✅ 按使用流程排序

### 避免事项

- ❌ 模糊或像素化的图片
- ❌ 过多的空白区域
- ❌ 包含敏感信息
- ❌ 使用虚假或误导性内容
- ❌ 过度编辑或添加效果

## 📊 可选: 宣传图片

### 小型宣传图 (440x280)

- 在 Chrome Web Store 列表中显示
- 简单明了地展示插件 logo 和名称

### 大型宣传图 (920x680)

- 在 Chrome Web Store 详情页显示
- 可以包含更多功能介绍

### Marquee 图 (1400x560)

- 用于特色推广
- 设计要求较高，建议使用设计工具

## 💡 提示

- 第一张截图最重要，会在搜索结果中显示
- 使用真实的使用场景，不要用 lorem ipsum
- 如果插件支持多语言，考虑准备多套截图
- 可以添加简短的文字说明突出功能点
- 定期更新截图以反映最新功能
